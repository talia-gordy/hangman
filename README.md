## Overview 
- This respitory contains two files labeled "hangman" and "wordlist", both are needed for the interactive game.

--------------------------------------------------------------------------------------------------------------------

## How to use?

1. Download "hangman" and "wordlist" located in this respitory.
2. Open a code editor.
3. Open both files and run the interactive hangman game, play with as many people you'd like to guess the word!
- **Note: This hangman game utilizes the random function indicated by "import random" Which chooses a random word from the file "wordlist" to allow the game
to be one player friendly!
